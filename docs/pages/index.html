Double

A CSS framework with dark mode

Get Double

Get Double framework by adding this style on your page 

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/emanuelescarabattoli/double@0.0.1/dist/double.min.css" />

Why Double?

Double allow you to implement the dark mode in you site just by adding the class dark-mode in the main container of a page, just like the following example 

<!doctype html>
<html>
<head>
  <meta charset="utf-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1" />
  <link rel="stylesheet" href="./assets/double.min.css" />
</head>
<body>
  <div class="dark-mode">
    <!--Your HTML code here-->
  </div>
</body>
</html>
